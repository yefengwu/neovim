# Neovim

## Installation

### linux

```bash
git clone git@github.com:yefengwu/neovim.git $HOME/.config/nvim

```

### Win

```powershell
git clone git@github.com:yefengwu/neovim.git HOME\AppData\Local\nvim
```
```nushell
git clone git@github.com:yefengwu/neovim.git ~\AppData\Local\nvim
```
